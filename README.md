# Nail Studio — сайт-визитка для салона маникюра

Это мой первый проект в портфолио. Сайт свёрстан с нуля на HTML и CSS.

## Ссылка на сайт
🔗 https://vasileva17.github.io/my-first-landing/

## Что умеет сайт
- Красивая шапка с логотипом
- Блок с ценами на услуги
- Галерея работ
- Адаптивный дизайн (смотрится на телефоне и компьютере)

## Технологии
- HTML5
- CSS3 (Flexbox)

## Как запустить локально
1. Склонируй репозиторий: `git clone https://github.com/vasileva17/my-first-landing.git`
2. Открой `index.html` в браузере
